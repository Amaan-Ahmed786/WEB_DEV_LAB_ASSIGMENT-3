# Lab 3 — Responsive Personal Portfolio Website

**Author:** Amaan Ahmed  
**Roll No:** 2501730305  

## Lab Title
Responsive Personal Portfolio Website (Flexbox, Grid, Media Queries, Animations)

## Description
One-page responsive portfolio demonstrating:
- Mobile-first CSS using Flexbox for layout and CSS Grid for content blocks.
- Responsive breakpoints for mobile / tablet / desktop.
- Hover/transition effects and a small keyframe animation.
- Accessible, semantic HTML structure and a sample contact form (static).

This repository contains the final files you can open locally or deploy to GitHub Pages.

## Files
- index.html — main one-page site (semantic sections: hero, about, skills, projects, contact).
- css/styles.css — all styling (Flexbox, Grid, media queries).
- Profile-Picture
